🛣️ Les circuits du World Grand Prix

Découvrez les 3 tracés mythiques du Grand Prix mondial, entre vitesse, décors spectaculaires et pièges inattendus !

🇯🇵 Tokyo

  • Longueur : ~4,5 km
  • Type : Urbain nocturne
  • Décor : Néons, tours, carrefour géant
  • Événement : Victoire de Francesco, crash de Shu
  • Inspiration : Shibuya, Tokyo Drift

🇮🇹 Porto Corsa

  • Longueur : ~5,1 km
  • Type : Côte méditerranéenne
  • Décor : Falaises, ruelles, marina
  • Événement : Victoire de McQueen
  • Inspiration : Monaco, Portofino

🇬🇧 Londres

  • Longueur : ~4,8 km
  • Type : Centre-ville + monuments
  • Décor : Big Ben, ponts, rues anglaises
  • Événement : Tentative d’attentat déjouée
  • Inspiration : Westminster, Londres réelle

Programmation

Depuis le début de ma formation en septembre, j’ai également commencé à explorer plusieurs langages de programmation et technologies web. Bien que je sois encore en phase d’apprentissage, voici les bases que j’ai commencé à acquérir :

  • Initiation au C# : J'ai découvert ce langage orienté objet souvent utilisé pour le développement d’applications Windows et de jeux, particulièrement avec le framework .NET.
  • Découverte de HTML : Apprentissage des bases du HTML, le langage de balisage standard pour créer des pages web, permettant de structurer le contenu et d'intégrer des éléments multimédias.
  • Apprentissage des bases de Python : Initiation à Python, un langage polyvalent et accessible, utilisé pour le développement web, le Data Science, et l'automatisation des tâches.
  • Introduction au PHP : Découverte de PHP, un langage de script utilisé principalement pour le développement web côté serveur, permettant de créer des sites dynamiques et interactifs.
  • Exploration de JavaScript : Familiarisation avec JavaScript, le langage de programmation essentiel pour rendre les pages web interactives et dynamiques, largement utilisé dans le développement front-end.
  • Utilisation de MySQL : Utilisation de requêtes MySQL dans des Bases de données.
